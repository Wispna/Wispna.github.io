Bot Commands

We will be using a variety of bots in the Curse of Raehn Discord to aid with combat, the store, and money/xp earning. As more bots and commands are added, more instructions will be added as well.

Quasi-bot



To Add a new Character:

!addchar <character name> : Note that every character name has to be unique, and cannot contain spaces. If there is a space, everything after the space will automatically be added to the character's description.

To Delete a Character:

!deletechar <character name> : Note that the character name has to be case sensitive when deleting. Use this command with caution.

Change image:

!charimage <charname> <img url> : it must be a url and cannot simply be uploaded along with the command. (don't actually put <> around the url as it will break the card)

Change character description:

!editchar <charname> <description>

Change magic type:

!charmagic <charname> <magic type>

Change stats:

!charstats <charname> 0 0 0 0 0 0 : Note that your replace the 0's with the corresponding stat number, and they must be in order of vitality > physical > sorcery > academics > lore > blessings (the same order they are on the card). There must also be spaces in between each separate stat number.

Change skills:

!charskills <charname> <character skills>

Other page skills:

!charskillsm <charname> ----- then press return <charskills> : Note that this is for magic skills

!charskillsp <charname> ----- then press return <charskills> : Note that this is for physical skills

To create a harvesting location you must have first bought a business license in the shop. People may then enter your business thread to harvest ingredients for craftable items.

!addlocation <locationName>: This will create the initial location where your items can be harvested from. The location name must be one word without spaces. Try to make it something simple to type.

!createitem <itemName>: This will create a custom item that can be potentially harvested from your location, or crafted using those harvested items. The name can contain spaces, and its properties can be edited in commands shown below.

!itemdescription <description>: This will create the description for your item that is viewed when your item is looked up with the !item command.

!itemtype <type>: This will explain what kind of item it is so it can be filtered when viewed in the inventory. Currently only consumables (potion), materials (material), miscellaneous (misc), and magic (magic) items are allowed to be created. Custom weapons/accessories/armor or anything that gives a permanent stat boost cannot be made.

Consumables are items that give a temporary boost, materials are items that can be harvested, miscellaneous is non magical misc items, and magic are items with rp flavor magical effects.

!itemimage <imageurl>: Do not actually include <> when inserting the url. This will make the url not work when accessed by the bot. This will provide the image for when the item is looked up.

!itememoji <emojiName>: This will add an emoji next to your custom item that can be viewed when the item is looked at in the inventory. This emoji MUST be from the Curse of Raehn server/must be a default discord emoji. Quasi is unable to access emojis from other servers.

!itemlocation <location name>: This is for harvestable materials only! The location name must be a location that you have already created in the bot, and must have the same capitalization. This will make it so that when the <!harvest> command is used, this item can be harvested from the right location.

!harvest <location name>: This command will harvest harvestable materials in the location. It will randomly choose from all of the materials that were added from that location. This command can only be used every 6 hours, and has a 20% chance of finding nothing. You cannot harvest from multiple locations each 6 hours, so one location must be chosen.

!itemingredient <ingredient number (1-5)> | <craftable item name> | <ingredient item name> | <amount of this ingredient needed>: This is for craftable items only. Craftable items may have up to 5 different ingredients. For example, if one wanted to craft an umbrella using 3 twigs and 2 cloth, then the commands would be:

!itemingredient 1 | umbrella | twig | 3

!itemingredient 2 | umbrella | cloth | 2

This would set the first ingredient needed as twig, and the second ingredient needed as cloth. Once this is set up, the item will be able to be crafted with the !craft command.

!iteminfo <itemname>: This command will show a simple profile card that will display the information that you inputted. It can show you what you have the location set as, as well as what you have the ingredients set as.

!uci <itemname>: uci stands for use custom item. This will delete a single one of an item in your custom item inventory.

!bag: custom user items will display in a different inventory. This command will show you your custom user items inventory, as opposed to your normal inventory.

!tci <@user> <itemname>: tci stands for transfer custom item. This will transfer an item from your bag to another user's bag.



This is a custom bot that will aid with much of the system and combat. Important CoR commands to know are:

!skill - using !skill <skill #> will roll the specified amount of d6's for you, count every 5 and 6, and will then let you know how many successes you got.

!init - using !init <phys #> will automatically add the results of your d6's to calculate your initiative.

!atk - !atk <atk #> <bless #> will calculate the successes, along with the blessing critical multiplier. It will then tell you the total amount of damage you managed to do.

!heal - !heal <sorc #> <bless #> This is the same as !atk, except it's for healing.

!def - !def <def #> <bless #> This is the same as !atk, except it's for defending.

!multiatk - !multiatk <atk #> <bless #> <monster amount> This lets you quickly attack with multiple monsters at once.

!profile - !profile <character name> or !char <character name> will let you view a small character profile card that lets you know their basic combat information.

!build - !build <buildingname> will let you add progress to building select buildings. Sometimes random events will happen. You can use this command every 12 hours.

!explore - currently only <!explore death> is available. This command can be used once per day and will reveal a section of Raehn's underground!

!search - !search <item name>. This command lets you look more closely at items that particularly stand out when exploring.

!buildings - Shows the build progress of and lists the buildable buildings.

!inv - Shows your character's in character inventory.

!balance - Shows how many glass sparrows you have. Glass sparrows can be earned through building, or through certain store items.

!balances - Shows how many glass sparrows everyone has.

!chars - Lists all the characters currently apped for Curse of Raehn.

!reps - Shows the reputation of every character.

!levels - Shows how much experience every character has earned.

!item - !item <item name> will give you info about that particular item.

!help - Shows extra bot commands.

!transfer - !transfer <@recipient> <amount> will transfer an amount of birds from your balance that you specify to a chosen person.

!transferitem - !transferitem <@recipient> <item name> will transfer a single item of your choice from your inventory to that of a chosen person's.

!useitem - !useitem <item name> will delete a single instance of the named item from your inventory.

!xppotion - !xppotion <character name> <# of potions> will use any xp potions in your inventory to give xp to a chosen character.


Unbelievaboat



Official website here: https://unbelievaboat.com/

This is a bot we will be using to manage much of our economy. It will handle rp and art earnings, as well as the server's store.

>store - This command will allow you to view the store.

>buy-item - >buy-item <item name> will allow you to purchase an item from the store.

>use-item - >use-item <item name> uses an item from the store. This mostly just deletes the item from your inventory.

>add-money @discordname <#> - This command will add money to your balance.

>remove-money @discordname <#> - This command will remove from your balance.
